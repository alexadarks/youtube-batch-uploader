# Contributing

Thanks for your interest in contributing to youtube-batch-uploader!

## How to Contribute

### Report Bugs
- Check if the issue already exists
- Provide:
  - Python version and OS
  - Steps to reproduce
  - Expected vs actual behavior
  - Error message / stack trace

### Suggest Features
- Describe the use case
- Explain why it would be useful
- Provide examples if applicable

### Submit Code

1. Fork the repo
2. Create a feature branch: `git checkout -b feature/your-feature`
3. Make your changes
4. Test: `python3 -m pytest` (if tests exist)
5. Commit: `git commit -am 'Add your feature'`
6. Push: `git push origin feature/your-feature`
7. Open a Pull Request

## Code Style

- Use Python 3.8+ syntax
- Format with PEP 8
- Add docstrings to functions
- Comment non-obvious logic

## Testing

If adding new features, please include tests or manual testing steps.

## Documentation

- Update README.md if changing behavior
- Add examples for new features
- Keep WORKFLOW.md and troubleshooting sections up-to-date
